# Cat Mode Extension

A delightful cat mode for E-NOR that transforms the robot into a feline companion with cat ears, whiskers, and authentic cat behaviors!

## Features

- **Cat Overlay**: Pointed ears and whiskers with subtle sparkles
- **Cat Sounds**: Meows, purrs, and other cat vocalizations 
- **Cat Behaviors**: Stretching, napping, and independent cat personality
- **Custom Emotions**: Content, sleepy, relaxed, curious, and playful cat expressions
- **Voice Triggers**: Natural phrases to activate cat mode and behaviors

## Voice Commands

- **"cat mode"** / **"activate cat mode"** - Enter cat mode
- **"deactivate cat mode"** / **"normal mode"** - Exit cat mode  
- **"meow"** / **"purr"** - Make cat sounds
- **"stretch like a cat"** - Do a cat stretch sequence
- **"nap time"** / **"cat nap"** - Take a short cat nap

## Files

- `manifest.json` - Extension metadata and voice triggers
- `handler.py` - Main cat behavior logic
- `overlay.svg` - Cat ears and whiskers overlay
- `emotion.json` - Custom cat emotions
- `sounds/` - Cat sound effects (meows, purrs)
- `README.md` - This documentation

## Sound Files

The following sound files should be added to the `sounds/` directory:

- `meow1.wav` - Short meow
- `meow2.wav` - Medium meow  
- `meow3.wav` - Long meow
- `purr1.wav` - Soft purring
- `purr2.wav` - Contented purring
- `meow_hello.wav` - Greeting meow
- `meow_goodbye.wav` - Farewell meow

## Cat Personality

When in cat mode, E-NOR becomes:
- More independent and selective in responses
- Less chatty than normal (cats are dignified!)
- Focused on comfort, stretching, and napping
- Playful but on their own terms
- Occasionally aloof but ultimately affectionate
